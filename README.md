# 📌 Regras e Organização do Grupo

Este documento define as diretrizes para garantir o bom funcionamento das reuniões e o andamento eficiente do projeto.

---

## 📅 Reuniões

- As reuniões ocorrerão **toda semana**, para alinhamento do projeto.
- Dia definido: **terça-feira, após a aula de PI**.
- Quem faltar **não poderá opinar** nas sugestões e tomadas de decisão.
- O **último a chegar** será responsável por registrar as informações (ATA).

---

## ⏰ Comprometimento

- Em caso de atraso, avisar o grupo com **12 horas de antecedência**.
- Quem atrasar nas entregas será responsável por **escrever a ATA da reunião**.
- Todos são responsáveis por:
  - Ler a ATA
  - Se organizar
  - Planejar suas tarefas para cumpri-la

---

## 📵 Conduta durante as reuniões

- Não utilizar o celular durante a reunião.
- Uso permitido apenas em casos de **complemento relevante**.
- Respeitar o momento de fala de cada integrante.

---

## 🤝 Trabalho em Equipe

- Ajudar os colegas sempre que necessário.
- Estar disponível para contribuir com o grupo quando alguém tiver dificuldades.
- O foco deve ser sempre o **resultado coletivo**.

---

## ⚖️ Tomada de Decisão

- Em caso de divergência de opiniões:
  - Será realizada uma **votação** para desempate.

---

## ✅ Responsabilidade Coletiva

Todos os integrantes são responsáveis pelo bom andamento do projeto e pelo cumprimento das regras estabelecidas.
